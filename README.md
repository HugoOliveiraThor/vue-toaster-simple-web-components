# vue-toaster-simple


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### How to use it 
```
- import the JS file to your project
- Before that follow the example 
```
<toaster 
    :params="{text:'Assinado com sucesso!', type:'success'}"
/>

```
```
